# ibig

[![crates.io](https://img.shields.io/crates/v/ibig.svg)](https://crates.io/crates/ibig)
[![docs.rs](https://docs.rs/ibig/badge.svg)](https://docs.rs/ibig)

A big integer library with good performance.

The library implements efficient arbitrary-precision integer arithmetic in pure Rust, built on
top of [`ibig-core`](https://crates.io/crates/ibig-core).

This is a from-scratch rewrite and is in early development.
